# ABCDEアプローチ参考情報

出典: https://emergency--nursing.com/2026/01/22/abcde/

## A：Airway（気道）
- 話せるか（声量、途切れ、嗄声）
- 異常な呼吸音：いびき様（舌根沈下）、ゴロゴロ（分泌物）、吸気性笛様音（狭窄）
- 胸腹部の逆運動（シーソー呼吸）
- 介入：体位調整、気道開通、口腔内吸引準備、高濃度酸素

## B：Breathing（呼吸）
- 呼吸数（増加傾向は悪化サイン）
- 呼吸の深さ/リズム/努力呼吸（肩で息、陥没、補助筋使用）
- チアノーゼ、冷汗、左右差
- SpO₂目標：一般94-98%、COPD 88-92%
- 介入：起坐位、酸素投与、RRT/医師コール

## C：Circulation（循環）
- 脈（速さ・強さ・規則性）
- 皮膚：冷感、湿潤、蒼白
- 毛細血管再充満、四肢末梢灌流
- 出血（ドレーン、創部、便・吐血）
- 介入：モニター装着、静脈路確保、採血準備、12誘導心電図

## D：Disability（意識・神経）
- 意識レベル（AVPU/GCS/JCS）
- 瞳孔（左右差、対光反射）
- 血糖（低血糖は即治療可能な致死的原因）
- 痙攣、麻痺、急な錯乱
- 介入：A/B/Cの是正優先、血糖是正

## E：Exposure（全身観察）
- 体温、発疹（アナフィラキシー/敗血症）、出血斑
- 外傷、熱傷、ライン/創部、浮腫
- 露出は最小限＋保温＋尊厳
